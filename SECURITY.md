# Security Policy

## Supported Versions

| Version | Supported                           |
| ------- | ----------------------------------- |
| 0.x     | ✓ (latest minor only, until 1.0.0)  |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub's [private vulnerability
reporting](https://github.com/jovandyaz/pycenote/security/advisories/new), or
by email to <jvaonam@me.com>. Do not file public GitHub issues for security
issues.

We aim to:

- Acknowledge within **7 calendar days**.
- Provide a fix, mitigation, or detailed analysis within **90 calendar days**.
- Credit the reporter (with consent) in the release notes.
